Общие сведения

Авторизация в системе

Для авторизации в системе перейти по адресу http://mysite.ru/login.html и ввести имя пользователя и пароль в форму авторизации

Структура интерфейса

После авторизации открывается система управления сайтом. Слева расположено меню вызова модулей системы, а справа формируется рабочее поле текущего вызванного модуля. Интерфейс работает без перезагрузки страниц. При работе с элементами в рабочем поле большинство функций управления вызывается при клике правой кнопкой на элементе. В браузере Operaиз-за особенностей системы безопасности вызов меню по правой кнопке невозможен, поэтому в опере меню управления элементом формируется при клике на дополнительно сформированный элемент в виде окружности со стрелками вниз:

Назначение пунктов меню:

  • Структура сайта – модуль управления страницами сайта
  • Изображения – модуль управления изображениями
  • Каталог – модуль управления каталогом товаров
  • Новости – модуль управления новостями
  • Блоки – модуль управления блоками
  • Пользователи – модуль управления пользователями

Работа с папками

Вид меню для папки:

Главное меню сайта формируется автоматически на основе страниц, размещенных в папке «Главное меню, русская версия сайта» Вспомогательные страницы, которые не отражаются в главном меню, необходимо размещать в папке «Дополнительное дерево сайта»

Для создания папки необходимо нажать на «+» и выбрать «Создать каталог», затем, щелкнув правой кнопкой мыши на появившемся каталоге, выбрать шаблон «Раздел», задать соответствующие права доступа.

Каждая папка может содержать страницы, для папки можно установить права доступа (видимости). Если какая-либо страница или папка существует в структуре сайта, но не отображается в меню и при доступе по прямому адресу ссылки система отображает сообщение, что страница не существует, то вероятнее всего не установлена видимость страницы в пункте «Права доступа» в меню для соответствующей группы пользователей. Для того, чтобы страница сайта была видна всем, нужно поставить права доступа "Анонимный пользователь".

Работа со страницами

Вид меню для страницы:

Создание страницы

Для создания страницы необходимо нажать на «+» и выбрать необходимый пункт меню. В зависимости от установленных шаблонов страниц, следует выбирать «Создать страницу на основе - Сайт. Главная страница», «Создать страницу на основе - Сайт. Внутренняя страница» или какой-то другой шаблон, если установлен.

По умолчанию главной страницей сайта является страница «Главная (index.html)» — она отмечена звездочкой, но при необходимости главной можно сделать любую другую страницу.

Изменение имени для меню и адреса ссылки

Для изменения названия для меню и адреса страницы, необходимо в меню страницы выбрать пункт «Переименовать», ввести новое имя страницы и нажать Enter.

После изменения имени страницы система предложит автоматически переведенный в транслит вариант ссылки страницы. При необходимости его можно переформулировать. При вводе допускаются только латинские символы, цифры и знак подчеркивания.

Заполнение метаданных – ключевых слов, описания, заголовка для браузера

Для изменения ключевых слов, описания и заголовка страницы для браузера, необходимо выбрать в меню пункт «Редактировать meta-информацию»

Редактирование содержимого страницы

Редактирование заголовка страницы

Если шаблоны страниц имеют возможность вывода заголовков для данных страниц на сайте, то эти заголовки можно изменять, выбрав соответствующий пункт меню.

Редактирование текста

Редактирование текста на странице (а также в новостях и информационных блоках) производится с помощью визуального редактора, по типу MicrosoftWord Для вызова редактора содержимого необходимо выбрать в меню страницы пункт «Редактировать текст»

Визуальный редактор

Внимание! Если вы хотите вставить текст, скопированный из MicrosoftWord или другого редактора, нажмите на кнопку «Вставить из Word» на панели редактора и вставьте текст в появившееся окно.

Вставка изображения

Для вставки в текст страницы изображения необходимо выбрать кнопку «Вставить изображение» на панели редактора. Появится окно вставки изображения. Можно сразу ввести адрес картинки, если он известен, либо посмотреть на сервере

При нажатии на кнопку «Посмотреть на сервере» откроется окно менеджера изображений CKFinder

Изображения в менеджере доступны для изменения, просмотра и вставки в текст редактора

Нажмите на кнопку «Загрузка» в панели менеджера, нажмите на кнопку «Выбрать файл» и выберите необходимое для загрузки изображение на своем жестком диске.

После того, как файл выбран, нажмите на кнопку «Загрузить выбранный файл»

После загрузки файл появится в менеджере, где его можно выбрать для вставки

Отредактировать дополнительные параметры, такие как ширина, высота, отступы текста от изображения, обтекание в тексте по левому или правому краю.

Вставьте изображение в текст редактора.

Вставка ссылки

Для вставки ссылки выделите текст, который хотите сделать ссылкой. Выберите в панели редактора кнопку «Вставить ссылку».

Откроется окно редактирования ссылки, где нужно ввести адрес для ссылки.

Нажмите «Ок» и выделенный текст станет ссылкой.

Если нужно удалить ссылку, выберите в панели редактора кнопку «Убрать ссылку».

Перемещение страницы

Перемещение внутри папки

Для перемещения страницы внутри папки (изменения порядка следования страниц внутри папки) её можно просто перетащить мышкой на желаемую позицию.

Перемещение между папками

Выбрать «Вырезать» в меню страницы

Перейти на один уровень выше папки, в которую необходимо вставить страницу и в меню целевой папки выбрать «Вставить»

Страница вставится внутрь целевой папки

Удаление страницы

Выбрать «Удалить» в меню страницы

При удалении страницы система запросит подтверждение удаления, для предотвращения случайного удаления. Нажмите либо «Отмена», либо «Удалить».

Работа с новостями

Создание новости

Для создания новости перейти в модуль новостей и выбрать «Добавить новость» в меню

В списке новостей появится новый элемент. В меню созданной новости выберите «Переименовать» и введите заголовок новости

Система сразу предложит автоматически переведенный в транслит адрес ссылки для новости. Можно изменить его

Редактирование текста новости

Для редактирования полного текста новости необходимо выбрать в меню новости пункт «Изменить текст»

Откроется визуальный редактор. Работа с ним аналогична работе с редактором страницы

Редактирование анонса новости

Для редактирования анонса новости необходимо выбрать в меню пункт «Изменить анонс»

Откроется поле редактирования анонса. Оно не содержит визуального редактора, так как предполагается, что анонс новости состоит из одного предложения текста.

Установка даты новости

Для установки отображаемой даты новости необходимо выбрать в меню пункт «Изменить дату»

В появившемся календаре выбрать желаемую дату отображения новости.

Удаление новости

Для удаления новости необходимо выбрать в меню новости пункт «Удалить». Появится окно подтверждения удаления.

Установка дополнительных параметров новости

Если предусмотрено вариантом сборки, возможно установить пометку «Важная новость», рядом с ней появится звездочка.

Если предусмотрено вариантом сборки, возможно дополнительно к анонсу присоединить изображение.

Появится окно в котором можно управлять каталогами и изображениями.

Нажав на «+» можно создать новый каталог или загрузить файл изображения.

Загрузив изображение, оно появится в целевой папке

Теперь его можно перетащить мышкой в серое поле справа, для прикрепления к новости. Для открепления изображения, перетащите его за пределы серого поля.

Нажмите «Выбрать».

Работа с модулем изображений

Модуль изображений служит для загрузки изображений на сайт.

Изображения можно добавлять как непосредственно в модуль изображений, так и создавая необходимое число каталогов для различных категорий изображений.

При выборе «Создать каталог» будет создан новый каталог, который можно переименовать, удалить или открыть.
При выборе «Загрузить файл» будет предложено выбрать необходимый файл на жестком диске компьютера. Выбрать можно как одиночный файл, так и несколько файлов.

После выбора, файлы загружаются на сервер и доступны для использования.

Работа с модулем управления каталогом товаров

Назначение модуля управления каталогом товаров

Модуль управления каталогом предназначен для создания различных категорий товаров, их подкатегорий и добавления в них различного рода товаров.

После создания каталога, он автоматически отображается в меню выбора категорий товаров на сайте. Перетаскивая мышью каталоги вверх-вниз, можно менять из расположение в меню.

Внимание! Изображения для каталога товаров должны быть загружены заранее в модуль «Изображения».

Создание каталога

В модуле «Каталог» нажатием на «+» выберите «Создать каталог». Появляется пустой каталог, который можно открыть, редактировать его описание, установить изображение каталога (только для каталогов первого уровня вложенности, не для подкаталогов), переименовать каталог или удалить его. В каталоге можно создать подкаталоги с подкатегориями товаров.

Добавление товара

В модуле «Каталог» нажатием на «+» выберите «Добавить товар». Появится «Новый товар», для которого можно редактировать описание, добавить или изменить изображение товара, переименовать товар или удалить его. Сортировать товары можно простым перетаскиванием мышью на нужное место. Возможно установить признак «Новинка» для товара.

Редактирование описания

При выборе «редактировать описание» будет открыто окно редактирования текста, в котором можно ввести необходимое описание товара. При вставке текста из MSWord обязательно используйте функцию «Вставить из Word». В противном случае, введенная информация будет отображаться некорректно из-за большого количества лишних служебных символов, копируемых из Word вместе с текстом.

Добавление изображения товара

При выборе пункта «Изображение товара» можно добавить, заменить или удалить изображение текущего товара. Появляется окно выбора изображения. Для того, чтобы добавить изображение к товару, это изображение необходимо изначально загрузить в модуль «Изображения», после этого оно станет доступно для выбора в модуле «Каталог». Найдите необходимое изображение, переместите его на серое поле справа и сохраните выбор. Изображение будет добавлено к товару. Для удаления изображения, перетащите текущее изображение за пределы серого поля и сохраните выбор.

Работа с информационными блоками

Назначение блоков

Информационные блоки содержат информацию, которая используется на многих страницах сайта, но меняется в системе управления в одном месте. Назначение каждого отдельного блока можно легко понять из его названия.

Редактирование содержимого блока

Для редактирования блока необходимо выбрать в меню блока пункт «Редактировать текст». Откроется визуальный редактор, такой же, как при редактировании страниц сайта

Настройки сайта

Изменение имени пользователя и пароля

Для изменения пароля пользователя системы управления, войдите в модуль настроек сайта, выберите раздел «Пользователи». Откроется список пользователей с описанием принадлежности к группе: Администраторов, Менеджеров, Гостей. Нажмите на необходимой учетной записи правой кнопкой мыши и выберите в выпадающем списке опцию «Изменить пароль» или «Изменить имя». Появится окно ввода нового пароля или имени. Будьте внимательны, запишите или запомните новый пароль пользователя сайтом, если вы его забудете, восстановление будет невозможно, так как пароли хранятся в базе данных в зашифрованном виде. В случае утраты пароля, свяжитесь с разработчиком.

Добавление пользователя

Чтобы добавить нового пользователя, выберите в модуле настроек сайта раздел «Пользователи» и нажмите на «+» и выберите в меню «Добавить пользователя»

Изменение логина и группы пользователя

Чтобы сменить логин, нажмите на текущем названии в поле «Логин»

Чтобы изменить принадлежность выбранного пользователя к группе, нажмите на название текущей группы в поле «Группа», появится выбор группы с соответствующими правами доступа к сайту: анонимный пользователь (только просмотр сайта), менеджер (редактирование страниц, текстов), администратор (редактирование страниц, доступ к настройкам сайта, управление пользователями).
